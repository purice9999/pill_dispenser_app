                        TROUBLESHOOTING GUIDE
================================================================================

ISSUE: Git not recognized
SOLUTION:
  1. Install Git from https://git-scm.com/download/win
  2. Add to PATH: C:\Program Files\Git\bin
  3. Restart terminal
  4. Try git command again

ISSUE: GitHub authentication fails
SOLUTION:
  1. Use GitHub CLI: gh auth login
  2. Or use GitHub Desktop app
  3. Or set up SSH keys
  4. Follow GitHub's authentication guide

ISSUE: Codemagic build fails
SOLUTION:
  1. Check build log for specific error
  2. Common issues:
     - Syntax error: Check getDiagnostics output
     - Dependency issue: Run flutter pub outdated
     - Android SDK: Should auto-install
     - Kotlin/Java: Already configured
  3. Fix issue locally and push again

ISSUE: APK won't install
SOLUTION:
  1. Ensure device has enough storage
  2. Try: adb uninstall com.example.pill_dispenser
  3. Then: adb install -r app-debug.apk
  4. Check device is in developer mode
  5. Try different USB cable

ISSUE: App crashes on startup
SOLUTION:
  1. Check logcat for error: adb logcat
  2. Common issues:
     - Database initialization error
     - Missing permission
     - Null pointer exception
  3. Fix in code and rebuild

ISSUE: BLE not connecting
SOLUTION:
  1. Check HM-10 is powered
  2. Check HM-10 is in pairing mode
  3. Check Bluetooth is enabled on device
  4. Try scanning again
  5. Check HM-10 firmware

ISSUE: Alarms not persisting
SOLUTION:
  1. Check SQLite database initialization
  2. Check database path is writable
  3. Check logcat for database errors
  4. Verify AlarmProvider.initDatabase() is called
